## Contentful を使用する前提で、.env が必要
``` .env
CONTENTFUL_SPACE_ID=[***]
CONTENTFUL_ACCESS_TOKEN=[***]
```

### Algolia のために .env に追記が必要
``` .env
GATSBY_ALGOLIA_APP_ID=[***]
GATSBY_ALGOLIA_SEARCH_KEY=[***]
ALGOLIA_ADMIN_KEY=[***]
GATSBY_ALGOLIA_INDEX_NAME=[***]
```
